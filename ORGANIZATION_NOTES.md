# /Volumes/2t-xx Organization Notes

This volume now groups common files into a few top-level folders:

- `archives/`: zipped/tar backups and project bundles.
  - `archives/html/backups/`: HTML backup snapshots (e.g., HTML_Backup_20251026_*).
  - `archives/media/video/`: video bundles (Ai-Art-Mp4).
- `reports/`: analysis and status reports (flattening, API keys, summaries), guides, and logs.
  - `reports/logs/misc/`: misc logs/JSON analyses (fluid_analysis_*, update-log-*, Canva compressed analysis, workflows scenarios).
  - `reports/analysis/transcripts/`: transcript-related assets.
- `scripts/`: shell helpers for setup, inventory, and automation.

Active project/content folders (for example `AvaTarArTs`, `ai-sites`, `avatararts-deployment`, `simplegallery`, etc.) remain in place. If you add new backups or reports, drop them into `archives/` or `reports/` to keep the root tidy. Scripts or new setup helpers should go in `scripts/`.
