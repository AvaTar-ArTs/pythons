# Tools Directory

This directory contains the core automation tools organized by function:

## 📊 Analysis Tools
Located in `analysis/` - Tools for code analysis, data analysis, and system analysis:
- Code complexity analysis
- AST-based code inspection
- CSV and data pattern analysis
- Dependency visualization
- Performance analysis

## 🧹 Cleanup Tools
Located in `cleanup/` - Tools for organizing and cleaning files:
- File organization
- Directory flattening
- Name standardization
- Content-based cleanup
- Batch processing

## ♻️ Deduplication Tools
Located in `dedup/` - Tools for identifying and removing duplicates:
- Content-based duplicate detection
- Name-based duplicate identification
- Safe duplicate removal
- Duplicate consolidation

## 🏷️ Rename Tools
Located in `rename/` - Tools for batch renaming operations:
- Version-based renaming
- Pattern-based renaming
- Batch rename execution
- Rename validation

## 🔍 Scanner Tools
Located in `scanners/` - Tools for scanning and analyzing files:
- Function definition extraction
- File property analysis
- Content scanning
- Metadata analysis
- Batch file scanning

## 📋 Naming Convention

Tools follow a naming convention where:
- Primary tool: `tool_name.py`
- Variants: `tool_name_1.py`, `tool_name_2.py`, etc.
- Enhanced versions: `tool_name_1_1.py`, `tool_name_1_2.py`, etc.

This structure allows for easy maintenance and evolution of tools while keeping similar functionality grouped together.
