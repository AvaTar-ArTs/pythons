## What is SimpleGallery 2.0?

SimpleGallery 2.0 is a **major upgrade** that combines:

1. **Modern Codebase** - Updated APIs, security improvements, better architecture
2. **Enhanced Features** - AI-powered content analysis, advanced metadata extraction
3. **Improved UI** - Modern design, search, filtering, analytics
4. **Backward Compatibility** - All existing galleries work without changes

## Installation

```bash
# Basic installation
pip install -r requirements.txt

# With AI features (optional)
pip install torch torchvision pytesseract
```

---

## Quick Start

### Standard Gallery
```bash
python -m simplegallery.gallery_init -p /path/to/gallery --use-defaults
python -m simplegallery.gallery_build -p /path/to/gallery
```

### Enhanced Gallery
```bash
python -m simplegallery.gallery_init -p /path/to/gallery --use-defaults
python -m simplegallery.enhanced_gallery_build --enhanced -p /path/to/gallery
```
