## Installation

```bash
# Install dependencies
pip install -r requirements.txt

# Set up environment variables (optional)
cp .env.example .env
```

## Usage

```bash
# Analyze a project
python advanced_code_analyzer.py /path/to/project

# Deduplicate files
python advanced_file_deduplicator.py --path /path/to/project --dry-run

# Optimize directory structure
python avatararts_directory_optimizer_agent.py --path /path/to/project --backup
```
