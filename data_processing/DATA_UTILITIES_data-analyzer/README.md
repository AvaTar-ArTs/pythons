# Data processing and analysis

## Overview
This directory contains 12 Python scripts related to data processing and analysis.

## Categories
- **Production Scripts**: Well-tested, production-ready code
- **Experimental Scripts**: Work-in-progress, testing, and experimental code
- **Utility Scripts**: Helper functions, tools, and utilities
- **AI Tools**: Machine learning, AI, and data science scripts
- **Media Processing**: Image, video, and audio processing scripts
- **Web Scraping**: Web automation, scraping, and API scripts
- **Data Analysis**: Data processing, analysis, and visualization scripts
- **Automation**: Workflow automation and batch processing scripts
- **Learning**: Educational and tutorial scripts

## File Organization
Files are organized by functionality and maturity level. Use the naming convention:
- `script_name.py` - Main script
- `script_name_v2.py` - Updated version
- `script_name_test.py` - Test version

## Usage
Most scripts can be run directly with Python:
```bash
python script_name.py
```

For scripts with dependencies, check the imports at the top of the file.
